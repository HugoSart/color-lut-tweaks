# Agents

### Folder Structure:
- ./instructions: custom instructions to include
- ./notes: notes to consider and previous exploration findings that are worth to be remembered
- ./skills: skills folder
- ./architecture: the application architecture GOAL; use it to know how the code structure / user UX looks like
- ./workspace: non-checked folder that you can use to dump temporary test scripts and other files to support your 
  current tasks:
  - temporary scripts
  - temporary checkouts
  - etc
